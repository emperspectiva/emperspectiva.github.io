Você sabe o que é um algoritmo?

Algoritmo é uma sequência de instruções que definem como uma tarefa qualquer deve ser executada. A tarefa pode ser tão simples quanto uma receita culinária ou tão complexa quanto pousar um foguete na lua.

Veja o algoritmo a seguir que poderia servir para um robô descer uma escada:

Enquanto houver degraus à vista:
		Caminhe para o degrau e desça.
		Se encontrar uma parede:
			Vire. 
	Finalize.

Essencialmente, programas de computador são compostos por algoritmos.
